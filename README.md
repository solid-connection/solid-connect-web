솔리드 커넥션 웹

React + Next.js

## Commands
```bash
npm run dev
```


## 파일 구조

- public: 정적 파일
- src
    - /components: 컴포넌트 파일
    - /context: 
    - /libs: 외부 라이브러리 사용 파일
    - /pages: Next.js 지정 page 파일
    - /services: API call 인터페이스
    - /styles: 정적 css 파일
    - /types: TypeScript 타입 정의, 상수 파일
    - /utils: 유틸성 함수
- .env: 공개 환경 변수
- .env.local: 미공개 환경 변수
- .eslintrc.json: ESLint 설정
